# AWS Cloud Architecture Design

## Overview

This project presents a scalable cloud architecture designed for deploying a web application on AWS.

The architecture focuses on high availability, scalability and security following AWS Well-Architected principles.

---

## AWS Services

- Elastic Beanstalk
- EC2
- Application Load Balancer
- Auto Scaling
- RDS
- VPC
- IAM
- CloudWatch
- Security Groups
